# PromptNest

PromptNest is a local-first Windows prompt library built with WinUI 3, .NET 8, SQLite, and MVVM.

This repository is currently being implemented from `IMPLEMENTATION_PLAN.md`.

## Current Status

The solution scaffold is in place. Feature implementation is tracked in Linear under the PromptNest UI Implementation project.

## Build

```powershell
dotnet restore
dotnet build PromptNest.sln
```
